# Stellar Structure Calculation 

Code that solves the equations of stellar structure to derive a pressure, temperature, luminosity, and radius dependence on mass integrating outwards from the core and inwards from the star's surface. Project is completed for the Stellar Structures and Evolution class final project at JHU, Spring 2026.

The final report can be found at stellar_structure_report.pdf. Final plots and a machine readable table for a 2 solar mass star are included in the directory plots_2 and within the final report.
