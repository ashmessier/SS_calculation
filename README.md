# Stellar Structure Calculation 

Code that solves the equations of stellar structure to derive a pressure, temperature, luminosity, and radius dependence on mass integrating outwards from the core and inwards from the star's surface. Code is completed for Stellar Structures and Evolution final project at JHU, Spring 2026.
